phpALUGen

About:

phpALUGen is an adapatation of the Visual Basic (VB) ActiveLock Universal Generator (ALUGEN) in PHP. It is intended to be used along with the ActiveLock software to generate product keys and demonstration keys for ActiveLock programs easily and effectively on the internet.

At phpALUGen's core, it is a program for manipulating product keys. It can import products.ini files from the VB ALUGEN and store products in standard files or in a MySQL database. The phpALUGen library also allows creation of product keys (to be embedded in ActiveLock software) and liberation keys (to allow ActiveLock software to be unlocked). phpALUGen also contains a number of interfaces such as the administration interface and the BitPass interface which allow for keys to be added and generated as necessary.

Documentation:

End-user Documenation: The end-user documentation does not cover everything that can be done with phpALUGen, but many of phpALUGen's features are self-explanitory, especially the administration interface. The flash-based tutorials were created with Wink.

Installing phpALUGen - a flash-based tutorial on how to install phpALUGen for a MySQL database. The tutorial assumes you have already extracted phpALUGen to a directory on a webserver.
Importing A products.ini File - a flash-based tutorial on how to create a product in the VB ALUGEN and import it into phpALUGen.
Generating A Liberation Key - a flash-based tutorial that details how to create a liberation key for a product using the administration interface.

Developer Documentation: The standard developer documentation is for developers who would like to use the phpALUGen library to create an interface for generating liberation keys, etc. The phpALUGen developer documentation is for developers who would like additional information on how phpALUGen works and need to work with the details of phpALUGen functions for generating keys.
